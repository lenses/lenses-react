## Quickstart 
### setup
1. clone repo
2. `npm install`
3. `gulp serve`   
    if gulp is not installed, `npm install -g gulp`

### add custom component
1. `gulp serve`
2. create a .jsx react file in app/components/custom
3. add component in ui using it's name
